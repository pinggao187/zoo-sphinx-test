Bases: zoo.feature.image3d.transformation.ImagePreprocessing3D

Affine transformer implements affine transformation on a given tensor. To avoid defects in resampling, the mapping is from destination to source. dst(z,y,x) = src(f(z),f(y),f(x)) where f: dst -> src :param affine_mat: numpy array in 3x3 shape.Define affine transformation from dst to src. :param translation: numpy array in 3 dimension.Default value is np.zero(3). Define translation in each axis. :param clampMode: str, default value is “clamp”. Define how to handle interpolation off the input image. :param padVal: float, default is 0.0. Define padding value when clampMode=”padding”. Setting this value when clampMode=”clamp” will cause an error.
